**5단계: 입고 및 출고 프로세스 (핵심 운영 데이터)** 

실제 물류 흐름에 따라 데이터가 발생합니다.

|                   |                                                                                   |
| ----------------- | --------------------------------------------------------------------------------- |
| 입고 요청(InRequest)  | 고객(User)이 특정 상품(Product)의 입고를 요청합니다.                                              |
| 입고 처리(Inbound)    | 입고 요청(InRequest)을 기반으로 실제 창고에 물건이 들어온 내역을 기록합니다. <br>이 때 Stock 테이블의 재고 수량이 변경됩니다. |
| 출고 요청(OutRequest) | 고객(User)이 상품의 출고를 요청합니다.                                                          |
| 출고 처리 (Outbound)  | 출고 요청(OutRequest)을 기반으로 실제 창고에서 물건이 나간 내역을 기록합니다. Stock 테이블의 재고 수량이 변경됩니다.        |
