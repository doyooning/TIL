## Programmers SQL

------------------------------------
**1. 특정 형질을 가지는 대장균 찾기**

```
SELECT count(*) as count  
FROM ecoli_data  
WHERE (genotype & 2) = 0 AND ((genotype & 1) > 0 OR (genotype & 4) > 0);
```
& : 비트 단위 AND 연산자  
 -> 두 정수 값을 2진수로 변환, 
 각 비트를 비교해서 둘 다 1일 경우 1 반환
 그렇지 않으면 0 반환

0001(2) = 1, 0010(2) = 2, 0011(2) = 3, 0100(2) = 4  
=> 1번 형질 = & 1, 2번 형질 = & 2, 3번 형질 = & 4

**2. 조건에 맞는 개발자 찾기**

```
SELECT id, email, first_name, last_name  
FROM developers  
WHERE skill_code & (  
    SELECT code  
    FROM skillcodes  
    WHERE name = 'C#'  
) > 0 OR skill_code & (  
    SELECT code  
    FROM skillcodes  
    WHERE name = 'Python'  
) > 0  
ORDER BY id ASC;
```
& 비트 연산자 -> (큰 수) & (작은 수) 했을 때  
1. 결과가 0 => 정확히 일치, 두 수가 같음  
2. 결과가 > 0 => 큰 수에 작은 수가 포함됨  
3. 결과가 = 0 => 포함 안 됨
