##### 트리거란?
데이터베이스에서 데이터 삽입, 변경, 삭제 등 특정 이벤트가 발생할 때마다 자동으로 실행되는 작업

데이터 일관성을 유지하는데 다양한 작업에 적용
ex) 데이터 검증, 보안, 로깅 등

##### 구성 요소

###### 1. 트리거 정의
`CREATE TRIGGER ...`

트리거 생성 확인
`SHOW 트리거명;`

트리거 삭제
`DROP TRIGGER 트리거명;`

###### 2. 이벤트
데이터베이스에서 발생하는 특정 동작 또는 상태 변화를 이벤트로 설정

###### 3. 테이블
트리거가 연결될 대상 테이블 지정

###### 4. 타이밍
트리거가 실행되는 시점 정의
